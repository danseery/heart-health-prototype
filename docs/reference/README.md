# Product Reference Docs

This folder keeps original HeartHealth AI product reference artifacts so future
planning and implementation work can use the same baseline.

- `HeartHealth_AI_Proposal.pdf` captures the original side-project proposal,
  monetization model, launch sequence, and long-term product direction.
- `HeartHealth_AI_Mockups.pdf` captures early website mockups for the landing
  page, assessment, results dashboard, content library, AI coach, and pricing.

These PDFs are reference material, not binding implementation specs. Current
product decisions should also account for Dr. Chen's latest guidance: keep the
app cardiology-specific, source-grounded, and focused on curated education
instead of broad general-health advice.
